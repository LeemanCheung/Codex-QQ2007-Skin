# 获取支持

## 使用问题

先查看 [安装文档](docs/INSTALLATION.md) 和 [故障排查](docs/TROUBLESHOOTING.md)。仍无法解决时创建 GitHub Issue，并提供：

- Windows 与 Codex 版本；
- Node.js 版本；
- 执行的命令和退出码；
- `verify.json` 中与布局相关的字段。

请先删除任务标题、项目名、用户目录、账号、用量、Token 数、端口/PID 和其他个人信息。不要上传完整运行日志、会话 JSONL 或 `.codex` 目录。

## 安全问题

请按 [SECURITY.md](SECURITY.md) 使用 GitHub Security Advisory 私密报告，不要公开可利用细节。

## 支持范围

项目仅支持最新公开代码与官方 Windows Codex 桌面应用。主题依赖未公开承诺稳定的 DOM；新版 Codex 导致的结构变化会尽力适配，但不保证即时修复。
