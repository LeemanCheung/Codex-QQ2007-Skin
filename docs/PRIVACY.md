# 隐私说明

## 项目本身读取的数据

| 数据 | 用途 | 是否持久化 | 是否由主题上传 |
| --- | --- | --- | --- |
| 当前任务标题 | 标题栏显示 | 否 | 否 |
| Agent 运行状态 | 在线/忙碌/离开展示 | 否 | 否 |
| “1 周”剩余百分比 | Q币余额展示 | 当前窗口最多缓存 6 小时 | 否 |
| 本地会话 Token 数值 | 计算等级 | watcher 内存与本机诊断字段 | 否 |
| Codex 版本、PID、回环端口 | 启动与安全校验 | 本机 runtime 状态 | 否 |

主题不读取消息正文用于业务处理。Token 统计器只读取 JSONL 文件尾部的统计事件，并只把累计数值、会话数量和公式标识交给页面。

## 不读取或不修改

- `.codex/config.toml`；
- Codex 聊天数据库内容；
- 浏览器 Cookie、密码或第三方账号；
- Codex 安装包与签名；
- 局域网或公网服务。

## 本机诊断

`%LOCALAPPDATA%\QQ2009ProgrammerCodex\runtime` 可能包含任务标题、项目名、版本、用量、Token 总数、端口、PID 和绝对路径。它不会被仓库收集，但用户不应原样上传。公开报告前仅保留问题所需字段并脱敏。

## 网络

主题素材以内嵌 `data:` URL 载入，不发起外部网络请求。CDP 仅绑定 `127.0.0.1`。这不改变 Codex 应用自身按其产品设计进行的网络通信；本项目无法代表或控制 OpenAI 的数据处理。Codex 产品信息与政策应以 OpenAI 官方说明为准。

## 删除

运行恢复脚本可停止主题 watcher、关闭调试会话并恢复官方外观。安装包和诊断日志默认保留，便于排障；用户可在确认路径后手工移除主题自己的 `%LOCALAPPDATA%\QQ2009ProgrammerCodex` 目录，但不要删除其 Junction 目标或 Codex 用户数据。
