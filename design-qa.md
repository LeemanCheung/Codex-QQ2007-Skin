# 视觉与交互验收摘要

唯一视觉基准为“Codex 2007”参考图；公开概念帖仅用于追溯灵感来源。仓库不提交实机截图，因为其中可能出现任务名称、项目名称和账号用量。

## 对照区域

| 区域 | 实现 | 验收重点 |
| --- | --- | --- |
| 标题栏 | 蓝色渐变、企鹅、动态任务标题、窗口按钮图 | 41px 可见高度，原生菜单隐藏 |
| 工具栏 | 新建任务、已安排、插件、站点、拉取请求、聊天 | 六项均映射到原生入口 |
| 左侧栏 | 用户卡、功能入口、任务/项目列表、在线状态和搜索 | 原生任务与项目行为保留 |
| 中央区 | 文档式消息、代码块、原生输入区 | 输入、模型和发送控件完整 |
| 右侧栏 | 好友卡、等级图标、好友展示区 | 用量/状态只显示可验证值 |
| 状态栏 | 安全、网络、状态、等级和时钟 | 无水平溢出 |
| 设置页 | 自动暂停主题 | 所有当前服务行可见且含原生图标 |

## 最近一次实机结果

- Codex：`26.715.7063.0`
- 视口：1280 × 820，DPR 1.5
- 正常页：`pass=true`、`nativeAppIntact=true`、发送命中原生控件、水平溢出为 0
- 设置页：`pass=true`、主题 class/合成节点均撤出、18 个当前服务行均有原生图标

这些结果来自本机运行态验证，不代表未来 Codex 版本的兼容保证。可复现流程见 [docs/VERIFICATION.md](docs/VERIFICATION.md)。
