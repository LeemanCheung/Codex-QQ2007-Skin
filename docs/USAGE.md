# 使用说明

## 日常启动与退出

- 从“QQ2009 程序员版 Codex”快捷方式进入皮肤版。
- 从官方 Codex 图标进入时保持官方外观。
- 使用“恢复原版 Codex”关闭主题、守护进程和 CDP 端口。

## 界面语义

| 原生 Codex | 主题显示 |
| --- | --- |
| 当前任务 | 好友消息 |
| 个人资料 | Codex 程序员资料卡 |
| 剩余用量中的 1 周百分比 | Q币余额 |
| Agent 状态 | 在线 / 忙碌 / 离开 |
| 任务完成通知 | 好友上线 |
| Git 提交/发送动作 | 发送消息 |
| 累计本地 Token | QQ等级与进度 |

六项工具栏动作、新建任务卡、左侧任务/项目、模型选择、附件和发送均通过原生控件完成。可见 QQ 按钮不复制业务逻辑；它覆盖或代理到原生点击目标。

## Q币余额

打开左下角个人资料菜单，再进入“剩余用量”。主题只捕获“1 周”的真实剩余百分比，并在当前窗口缓存最多 6 小时。读取不到时显示 `--`，不会把累计 Token、计划限额或其他周期冒充余额。

## QQ等级

统计器扫描本机 Codex 的 `sessions` 与 `archived_sessions` JSONL 尾部，只提取 `total_token_usage.total_tokens` 数值并按会话 UUID 去重。等级每 60 秒刷新；无可验证统计时显示待同步。

等级图标使用四进制组合：

- 星星：1
- 月亮：4
- 太阳：16
- 皇冠：64

例如 56 = 3 × 16 + 2 × 4，因此显示 3 个太阳和 2 个月亮。

## 设置页面

进入设置后主题会自动暂停，恢复官方设置导航和服务图标。返回应用后主题重新挂载。若设置页仍显示 QQ 标题栏，请先运行恢复脚本，再使用主题快捷方式重新启动，并参照 [TROUBLESHOOTING.md](TROUBLESHOOTING.md)。

## 运行态文件

本机诊断位于 `%LOCALAPPDATA%\QQ2009ProgrammerCodex\runtime`。这些文件可能包含版本、任务标题、用量、端口和 PID，不应直接发布或附到公开 Issue；请按 [SUPPORT.md](../SUPPORT.md) 脱敏。
